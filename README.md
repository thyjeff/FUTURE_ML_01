# AI-Powered Sales Forecasting Dashboard (Task 1)

## Project Overview
This project focuses on predicting future sales trends using historical data and Machine Learning.

## Features
* **Time Series Forecasting:** Used Facebook Prophet to predict sales for the next 90 days.
* **Data Visualization:** Visualized trends and seasonality using Matplotlib.
* **Tech Stack:** Python, Pandas, Prophet.
